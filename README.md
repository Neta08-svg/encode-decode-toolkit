# Free Encode / Decode / Hash API

Free encode/decode and hash API - base64, URL, hex, HTML entities, MD5/SHA, JWT decode.

**Live tool & docs:** https://encode.wrapper-agency.com

## Quickstart - free API, no key required

```bash
curl "https://encode.wrapper-agency.com/api/v1/encode?op=base64-encode&value=Hello"
```

Clean JSON, open CORS, no signup. Try the interactive tool and full docs at https://encode.wrapper-agency.com
